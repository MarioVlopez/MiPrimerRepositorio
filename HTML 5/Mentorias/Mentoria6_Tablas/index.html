Tablas en HTML - CSS

En HTML, las tablas se definen mediante la etiqueta <table>, que actúa como contenedor principal. Dentro de <table>, se utilizan <tr> para crear filas y <td> para definir celdas de datos. Para encabezados de columna, se emplea la etiqueta <th>, que se muestra en negrita y centrada por defecto. La etiqueta <thead> se utiliza para agrupar el contenido de los encabezados de la tabla, mientras que <tbody> se usa para agrupar las filas de datos, facilitando la organización y estilización. También se puede usar <tfoot> para agrupar el contenido del pie de la tabla, lo que es útil para totales o resúmenes. Adicionalmente, se puede incluir la etiqueta <caption> para proporcionar un título descriptivo a la tabla, mejorando su accesibilidad y comprensión. Con estas etiquetas, se puede crear una estructura de tabla clara y semánticamente correcta.
